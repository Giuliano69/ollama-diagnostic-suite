# Ollama Diagnostic Suite (ODS)

A lightweight diagnostic framework for testing, benchmarking and troubleshooting Ollama-based AI models.

## Mission

Ollama Diagnostic Suite (ODS) is an open-source toolkit designed to help developers understand **how** a model behaves, not only **how fast** it is.

ODS focuses on operational diagnostics such as:

- runtime stability

- memory usage

- GPU/iGPU behaviour

- context growth

- Ollama runtime monitoring

- repeatable benchmark scenarios

Rather than competing with benchmark suites, ODS complements them by helping identify instability, resource exhaustion and runtime anomalies before deploying a model into production.

## Design Principles

ODS follows a few simple principles:

- Measure before optimizing.

- Every benchmark must be reproducible.

- Change only one variable per experiment.

- Prefer stability over raw performance.

- Documentation is part of the project.
